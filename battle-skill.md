# 技能设计文档

> 本文档定义了 Keco Studio 回合制战斗系统的 30 个确定性技能，基于 `battle-design.md` 的战斗规则设计。
>
> **核心原则**：100% 确定性触发，无随机、无概率，满足条件必触发反应。

---

## 一、平衡性设计原则

### 1.1 元素强度标准

| 强度 | 符号 | 持续回合 | 衰减率 | 适用场景 |
|------|------|----------|--------|----------|
| 弱 | 弱 | 2 回合 | 10%/回合 | 普攻、低阶技能 |
| 中 | 中 | 3 回合 | 15%/回合 | 普通技能 |
| 强 | 强 | 4 回合 | 20%/回合 | 高阶技能 |

### 1.2 MP 消耗公式

```
MP消耗 = ceil(基础伤害倍率 × 基础系数 × 元素强度系数 × 效果系数)
```

| 参数 | 数值 | 说明 |
|------|------|------|
| 基础系数 | 10 | 1.0×ATK 技能的基准 MP |
| 元素强度系数 | 弱 1.0 / 中 1.1 / 强 1.2 |
| DOT 效果系数 | +1.2 | 附带持续伤害时 |
| 治疗效果系数 | -0.8 | 附带治疗时降低 MP |
| 控制效果系数 | +1.3 | 附带冻结时 |

### 1.3 冷却时间设计

| 技能倍率 | 建议冷却 | 说明 |
|----------|----------|------|
| 1.0 - 1.2× | 0 - 2 回合 | 低阶技能 |
| 1.3 - 1.6× | 3 回合 | 中阶技能 |
| 1.7 - 2.0× | 4 - 5 回合 | 高阶技能 |
| 2.0×+ | 5 - 6 回合 | 终极技能 |

## 二、普攻技能（3个）

> 特点：无 MP 消耗、无冷却，作为技能体系的基准线。

| # | 技能名 | MP | 冷却 | 伤害 | 元素附着 | 反应触发 | 说明 |
|---|--------|-----|------|------|----------|----------|------|
| 1 | **猛击** | 0 | 0 | 1.0×ATK | 无 | — | 纯伤害，无附加效果 |
| 2 | **元素戳刺** | 0 | 0 | 1.0×ATK | 弱随机(2回合) | 视元素而定 | 普攻附带元素 |
| 3 | **蓄力斩** | 0 | 0 | 1.0×ATK | 弱火(2回合) | 超载/燃烧 | 火系普攻 |

**设计理由**：
- 猛击作为纯伤害基准，玩家无资源压力时的选择
- 元素戳刺提供元素附着的可能性，但随机性由玩家接受
- 蓄力斩固定火元素，保证策略可预测性

## 十一、设计总结

### 11.1 核心特性

| 特性 | 实现 |
|------|------|
| 100% 确定性 | 所有效果触发无概率，满足条件必执行 |
| 元素反应必触发 | 技能设计中明确标注可触发反应 |
| MP 消耗平衡 | 基于伤害倍率×元素强度×效果系数计算 |
| 冷却设计合理 | 高伤害=长冷却，低伤害=短冷却 |
| 控制有代价 | 冻结/治疗类技能降低伤害倍率 |

### 11.2 策略深度

1. **元素搭配**：火+水蒸发、火+冰融化、雷+水感电等
2. **CD 管理**：高阶技能需要等待，使用时机重要
3. **MP 管理**：MP 有限，需要规划技能释放顺序
4. **先手优势**：SPD 决定谁先触发反应
