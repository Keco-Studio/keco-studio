# 战斗系统 UI 设计文档

> 本文档定义了 Keco Studio 回合制战斗模拟器的 UI 设计方案，基于 `battle-design.md` 和 `battle-skill.md` 的规则。

---

## 一、设计概述

### 1.1 设计目标

- **清晰展示属性**：HP、ATK、DEF、SPD、**MP**
- **直观选择技能**：30 个技能分类展示，清晰显示消耗和效果
- **实时状态显示**：元素附着、MP 剩余、DOT 状态
- **详细战斗日志**：元素反应、伤害计算、状态变化

## 三、交互设计

### 3.1 战斗流程

```
┌─────────────────────────────────────────────────────────────┐
│                         战斗流程                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [设置属性] → [选择敌人预设元素] → [开始战斗]                 │
│       ↓                                                    │
│  [玩家回合] → [选择技能] → [执行技能] → [触发反应]           │
│       ↓                                                    │
│  [敌人回合] → [AI选择技能] → [执行技能] → [触发反应]         │
│       ↓                                                    │
│  [回合结算] → [DOT伤害] → [MP恢复] → [状态衰减]               │
│       ↓                                                    │
│  [判断胜负] → [结束 / 继续]                                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 技能选择交互

1. **点击元素标签**：筛选对应元素技能
2. **点击技能卡片**：
   - 如果 MP 充足且冷却为 0 → 选中技能（蓝色边框）
   - 如果 MP 不足 → 显示提示"MP 不足"
   - 如果冷却中 → 显示提示"冷却中，剩余 X 回合"
3. **点击"使用技能"按钮**：执行选中的技能

### 3.3 技能快捷键

| 按键 | 功能 |
|------|------|
| 1-6 | 选择对应元素标签 |
| Q/W/E/R/T | 快捷选择前 5 个技能 |
| Space | 使用选中的技能 |
| Esc | 取消选择 |

## 六、状态管理

### 6.1 React 状态结构

```typescript
interface BattleState {
  // 属性配置
  playerConfig: CombatantConfig;
  monsterConfig: CombatantConfig;
  
  // 战斗状态
  battlePhase: 'setup' | 'player_turn' | 'enemy_turn' | 'round_end' | 'finished';
  currentTurn: number;
  
  // 战斗单位实时状态
  player: BattleUnit;
  monster: BattleUnit;
  
  // 技能相关
  selectedSkill: Skill | null;
  skillCooldowns: Record<string, number>;
  
  // 日志
  battleLogs: BattleLogEntry[];
  
  // 结果
  battleResult: 'player_win' | 'monster_win' | 'draw' | null;
}

interface BattleUnit {
  hp: number;
  maxHp: number;
  mp: number;
  maxMp: number;
  element: ElementAttachment | null;
  dot: DotStatus | null;
  buffs: Buff[];
}

interface ElementAttachment {
  element: Element;
  strength: ElementStrength;
  remainingTurns: number;
}

interface DotStatus {
  damage: number;
  remainingTurns: number;
}

interface Buff {
  type: 'atk_up' | 'def_up' | 'spd_up' | 'enflame' | 'electrified';
  value: number;
  remainingTurns: number;
}
```

### 7.2 字体

- 标题：`"Noto Sans SC", "PingFang SC", sans-serif`
- 正文：`"Noto Sans SC", "PingFang SC", sans-serif`
- 日志：`"JetBrains Mono", "Fira Code", monospace`

### 7.3 动画

| 动画 | 时长 | 效果 |
|------|------|------|
| 伤害数字弹出 | 600ms | scale 0→1.2→1, opacity 0→1→0 |
| HP/MP 变化 | 300ms | width transition |
| 元素附着 | 400ms | fadeIn + slight bounce |
| 元素反应 | 500ms | flash + particle effect |
| 技能选中 | 200ms | border glow |
